Heimdroid

Android app for heimcontroljs

Installation

  • Get the newest version of heimcontroljs (newer than 01/01/2014)
  • Download and install app, point your settings url to your Heimcontroljs Server (before you try to log in) and you are good to go.
  • Install apk via adb, download: heimdroid.apk

Note: I had to enable port 8080 or whatever on my local firewall, otherwise requests didn't go through. If you have your Server listening on port 80, you can don't need to specifiy the port. (i.e http://heimcontrol:80/ --> http://heimcontrol/)

Known bugs, will be fixed soon

  • Currently the heimcontroljs api doesn't return the initial toggle state when fetching all toggles.

For bugs and feature requeusts, please open a github issue.
